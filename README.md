# MakeItSo

In this application you can create a to-do list. I used it to study more about integration with Firebase and SwiftUI.

## Requirements
- iOS 16+
- Xcode 14
- Swift 5

## This project uses:
- SwiftUI
- MVVM
- Swift package manager
- Dependency Injection
- Factory
- Firebase
- Google Sign-In
- Apple Sign-In
